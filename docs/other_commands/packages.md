# pkg install

There are some commands for installing packages using [`npm`](https://npmjs.org) and [`pip`](https://pip.pypa.io).

```
> npm install
Package name?
ts
...
...
...
> 
```

*Installing TypeScript using npm*



### Installing packages

Use `pip install` and `npm install`

### Upgrading packages

Use:

- `npm upgrade`
- `npm update`
- `npm install -U`
- `pip upgrade`
- `pip update`
- `pip install -U`

## Parameters

No parameters
