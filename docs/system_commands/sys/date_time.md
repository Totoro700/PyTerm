# date time

The `date` **or** `time` command displays the date and time

```
> date
2020-12-15 08:28:21.349324
```

It will display something like

*`[year]-[month]-[day] [hour]:[minute]:[second].[millisecond]`*



```
> time
2020-12-15 08:28:21.349324
```

Both `time` and `date` produce the time and date
